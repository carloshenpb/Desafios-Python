# Manipulador de CSV (Python)

## Mini Projeto com objetivo de aprender sobre manipulação de arquivos .csv com python
### Objetivos do projeto:
    -> Criar arquivos .csv de maneira personalizada (Escolhendo quantidade e nome das colunas).
    -> Adicionar linhas a arquivos .csv.
    -> Manipular linhas especificas de arquivos .csv.
### Materiais utilizados para esclarecer duvidas:
    -> Cofre pessoal do Obsidian. 
    -> I.A (Claude e Gemini).
    -> Tutoriais no youtube.

### Funcionamento do código principal:
    -> Criação de arquivo:
        Opção 1 (Criar arquivo) -> Nome que o arquivo terá -> Verificar se existe algum arquivo com o mesmo nome -> Definir número de colunas -> Definir nomes das colunas -> Criar arquivo.
    -> Adicionando linha:
        Opção 2 (Editar arquivo) -> Opção 1(Adicionar linha) -> Nome do arquivo a ser editado -> Número de linhas a serem adicionadas -> Digitar valores que deseja adicionar.
    -> Editando Linha:
        Opção 2 (Editar linha) -> Opção 2(Editar linha) -> Nome do arquivo a ser editado -> Linha que será alterada -> Digitar novos valores.
